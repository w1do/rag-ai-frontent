## Описание проекта
- Фронтенд часть проекта для автоматизации и обучение RAG на различных данных, голосовые сообщения, JSON данные, парсинг сайтов, загрузка документов
- Система помогает быстро за 5 минут создать простого бота, для внедрения на сайт в виде виджета, или полноценное управление API на своей стороне

### Архитектура
Проект перешел от дробления к монолитной структуре основных страниц для обеспечения стабильности верстки:
- `src/components/WelcomeContent.astro` — Содержит все секции главной страницы в одном файле.
- `src/components/layout/` — Компоненты макета (Header, Footer, Sidebar, Meta).

### Структура главной страницы (WelcomeContent)
1. **Hero** — Главный экран с формой ввода.
2. **Image Slider** — Слайдер с примерами интерфейса.
3. **Features Micon** — Ключевые технологические преимущества.
4. **Features Generator** — Возможности обучения ИИ-агента.
5. **Why Section** — Преимущества платформы BotSync.
6. **Projects** — Сферы применения и кейсы (RAG).
7. **Marquee** — Бегущая строка с ключевыми технологиями.
8. **How It Works** — 3 шага к созданию своего ИИ-бота.
9. **Video** — Видео-презентация возможностей.
10. **Testimonials** — Отзывы клиентов об использовании ИИ.
11. **Pricing** — Тарифные планы.
12. **FAQ** — Часто задаваемые вопросы.
13. **Team** — Команда экспертов.
14. **Contact** — Форма обратной связи.

### Кому подойдет
> Подходит всем видам бизнеса, кому необходимо создать умного чат-бота, с детальным погружением в диалог
