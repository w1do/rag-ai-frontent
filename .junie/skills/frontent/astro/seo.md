### SEO Инструкции

Цель — максимальная видимость в поисковых системах для целевой аудитории проекта.

#### 1. Мета-данные

Для каждой страницы должны быть определены:

- `title`: 50-60 символов, содержит ключевые слова (Рыбалка, Кемпинг, Название локации).
- `description`: 150-160 символов, краткое описание контента.
- `og:image`: Превью страницы для соцсетей.
- `canonical`: Ссылка на основную версию страницы.

#### 2. Семантика и Структура

- **H1**: Только один на странице. Содержит главную тему.
- **H2-H3**: Логическое деление контента.
- **Alt-теги**: Обязательны для всех изображений. Описание должно быть релевантным (например, "Улов щуки на озере Байкал").
- **Списки**: Использование `<ul>` и `<ol>` для перечислений (преимущества, FAQ).

#### 3. URL Структура

- Человекопонятные URL (ЧПУ).
- Пример: `/locations/ozero-baikal`, `/reports/shchuka-na-spinning`.
- Использование дефисов в качестве разделителей.

#### 4. Оптимизация Изображений

- Использовать встроенный компонент Astro `<Image />`.
- Форматы: `.webp` или `.avif`.
- Указание `width` и `height` для предотвращения Layout Shift (CLS).

#### 5. Техническое SEO

- Генерация `sitemap.xml`.
- Настройка `robots.txt`.
- Использование JSON-LD (Schema.org) для:
  - Событий (Event)
  - Вопросов-ответов (FAQPage)
  - Локаций (Place)
  - Статей (Article)
